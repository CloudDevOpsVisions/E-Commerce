PBI Number	: 161

PBI Title	: Display special offer products in a slider

Primary Actors	:
	1. Visitor

Preconditions	:
	1. Three products was chosen by an admin to be displayed in the slider.

Basic Flow (Happy Path):
	1. Visitor navigates to the home page.
	2. A slider shows the first offer with buttons for navigation is displayed.
	3. Visitor clicks on a button for the second special offer.
	4. The second offer is displayed to the visitor.
	5. Visitor clicks on a button for the third special offer.
	6. The third offer is displayed to the visitor.

	
Postconditions	:
	None

Alternative Flow:
	None
