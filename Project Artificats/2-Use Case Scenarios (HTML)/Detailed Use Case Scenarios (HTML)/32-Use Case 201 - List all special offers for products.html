201 - List all special offers for products

Primary Actors

  • Vendor

Preconditions

  1. Vendor has been logged in to his account.
  2. Vendor has already added at least one offer.

Basic Flow (Happy Path)

  1. Vendor navigates to the vendor panel.
  2. Vendor clicks on list special offers link.
  3. The special offers list page is displayed, showing the first 20 special offers in rows ordered by recent special offer start date.
  4. Each row shows a single offer details which is Title, Product title, creation date, a delete icon, start and end dates.
  5. The Offer  title is also a link to update the offer.
  6. There is a paging area where vendor can navigate between different pages which displayes the rest of the offers.

Postconditions

None

Alternative Flow

3.1 The number of offers per page is set to 30

if the settings were modified to show 30 offers per page, then

3.1.1 The list will show 30 offer.

3.1.2 The use case resumes at step 4.
