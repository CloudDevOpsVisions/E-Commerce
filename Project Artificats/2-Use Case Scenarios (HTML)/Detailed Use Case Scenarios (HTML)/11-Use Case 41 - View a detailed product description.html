41 - View a detailed product description

Primary Actors 

  • Visitor

Preconditions

  1. A vendor has already added at least one product.

Basic Flow (Happy Path)

  1. Visitor navigates to the home page.
  2. Visitor clicks on a specific product from featured products slider.
  3. Another page for viewing the product is displayed.
  4. The page shows all details for the specified product, which are price, image, title, description, discount, stock availability, key features, and a fully detailed specifications.
  5. The page also has an add to cart button.

Postconditions

None

Alternative Flow

None
