Order		: 3

Id		: 28

Title		: View top new arrival products in a slider

Tags		: BWF, Public, Visitor

Description	: 
		As a visitor, I want to view top new arrival products in a slider on the home page so that I can buy popular 
		products that were unavailable for purchase for a long time.

Acceptance Criteria	:
		
	Action Criteria	:	
		* Three products at a time in slider's page should be displayed.
		* Two buttons for moving to next three and previous three product in slider should be included.
		* Three products should be loaded in advance from current one (next to current page and previous to current page).
		* On click on any product from slider, detailed view for that product should be displayed.
		* Limit number of pages in slider to be maximum 5 (it could be only one page if database does not have enough 
		  featured products), last item of 5th page should be a link titled "View All".
		* Last item of last page of slider ("View All") should open all new arrival products on a separate view/page 
		  where new products from each category will be displayed.
