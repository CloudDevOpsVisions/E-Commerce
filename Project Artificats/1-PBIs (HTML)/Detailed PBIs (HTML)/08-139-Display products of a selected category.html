Title: Display products of a selected category

Tags: BWF, Public, Vistor

Description:

As a visitor, I want to select a main or sub category from category list so that I can see what are the products in that category.

Acceptance Criteria:

Action Criteria:
  • All related products of selected category should be displayed in a list.
  • 20 products/page should be displayed.
  • There should be paging buttons to move to other pages.
