59 - Review order before checkout

Primary Actors 

  • Visitor

Preconditions

  1. Visitor has already added at least one item into his shopping cart.

Basic Flow (Happy Path)

  1. Visitor navigates to his shopping cart, confirms his address and clicks on review order.
  2. The order review page shows the total price, gift wrap information, and a list of all ordered items into rows.
  3. Each row shows a Product Image, Title, Quantity, Price and an icon to remove it.
  4. The visitor's personal information is shown, which is First Name, Last Name, Email, Phone, City, State, Postal Code, Country and Address.
  5. The page shows a button to edit the address and other personal information, a button to continue shopping and a button to continue to checkout.

Postconditions

None.

Alternative Flow

None.
