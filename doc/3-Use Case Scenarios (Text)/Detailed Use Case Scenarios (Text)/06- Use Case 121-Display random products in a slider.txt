PBI Number	: 121

PBI Title	: Display random products in a slider

Primary Actors	:
	1. Visitor

Preconditions	:
	1. Vendors have already added at least four products.

Basic Flow (Happy Path):
	1. Visitor navigates to the home page.
	2. A slider with one random product and buttons for navigation between 
	   next and previous random products is displayed.
	3. Visitor clicks on the next button.
	4. Another random product is displayed.
	5. Repeat step 3 and step 4 for three times.
	6. A link to view all products is displayed.

	
Postconditions	:
	None

Alternative Flow:
	None
