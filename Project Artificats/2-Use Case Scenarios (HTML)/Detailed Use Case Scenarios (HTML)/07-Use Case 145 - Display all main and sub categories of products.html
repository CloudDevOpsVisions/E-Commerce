145 - Display all main and sub categories of products

Primary Actors

  • Visitor

Preconditions

  1. An admin has already added at least one main category and at least one sub category inside.

Basic Flow (Happy Path)

  1. Visitor navigates to the home page.
  2. A navigation panel shows the list of main categories is displayed.
  3. Each main category has an icon and an arrow to expand it.
  4. Visitor clicks on an arrow for a specific main category.
  5. The sub categories list for the selected main category expands under its main category.

Postconditions

None

Alternative Flow

None
