PBI Number	: 119

PBI Title	: Display top best selling products in a slider

Primary Actors	:
	1. Visitor

Preconditions	:
	1. Vendors have already added some products.

Basic Flow (Happy Path):
	1. Visitor navigates to the home page.
	2. A slider with a best selling product and navigation support between best selling products
	   is displayed.
	3. Visitor navigates to the next best selling product.
	4. Another best selling product is displayed.
	5. Repeat step 3 and step 4 for three times.
	6. A link to view all best selling products is displayed.

	
Postconditions	:
	None

Alternative Flow:
	None
