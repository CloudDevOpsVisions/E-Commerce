Order		: 2

Id		: 138

Title		: Display featured products in a slider

Tags		: BWF, Public, Visitor

Description	: 
		As a visitor, I want to see featured products in a slider that exist on the home page below special offer 
		product's slider so that I can know about featured products.

Acceptance Criteria	:
		
	Action Criteria	:	
		* Three product at a time in slider's page should be displayed.
		* Two button for moving to next three and previous three products in slider should be included.
		* Three products should be loaded in advance from current one (next to current page and previous to current page).
		* On click on any product from slider, detailed view for that product should be displayed.
		* Limit number of pages in slider to be maximum 5 (it could be only one page if database does not have enough 
		  featured products), last item of 5th page should be a link titled "View All".
		* Last item of last page of slider ("View All") should open all featured products on a separate view/page.
