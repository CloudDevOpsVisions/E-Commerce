Title: View order details

Tags: BWF, Private, Vendor

Description:

As a vendor, I want to be able to view a detailed order information in a separate view so that I can explore more about the selected order.

Acceptance Criteria:

Action criteria:

  • First Name, Last Name, Email, Phone, City, State, Postal Code, Country, Address of shipping details (delivery address) of the order maker should be included.
  • Image, Title, Quantity, Price, Discount, Total amount and Gift Wrap option of added products should be included.
  • Each of displayed details can be changed by clicking on it.
