PBI Number	: 121

PBI Title	: Display random products in a slider

Primary Actors	:
	1. Visitor

Preconditions	:
	1. Vendors have already added some products.

Basic Flow (Happy Path):
	1. Visitor navigates to the home page.
	2. A slider with a random product and navigation support between random products is
	   displayed.
	3. Visitor navigates to the next random product.
	4. Another random product is displayed.
	5. Repeat step 3 and step 4 for three times.
	6. A link to view all products is displayed.

	
Postconditions	:
	None

Alternative Flow:
	None
