200 - Add a special offer for product

Primary Actors

  • Vendor

Preconditions

  1. Vendor have been logged in to his account.
  2. Vendor has already added at least one product.

Basic Flow (Happy Path)

  1. Vendor navigates to the vendor panel.
  2. Vendor clicks on the add special offer link.
  3. The add special offer page shows input fields for Offer Title, Image, Product Name, Start and end dates is displayed.
  4. The page has a grey (unavailable) save button.
  5. Vendor enters few letters of a product in the name product name field.
  6. A list of product names that contains letters entered by the vendor shows in the product name field.
  7. Vendor clicks on a specific product name from the list.
  8. The vendor fills other input fields with the details.
  9. The save button becomes blue (available).
  10. The vendor clicks on the save button.
  11. The offer details are validated.
  12. The save button becomes grey (unavailable) again, indicating that the offer was saved.

Postconditions

  1. The offer is now visible to members and visitors.

Alternative Flow

11.1 The offer title is too long

if the offer title is more than 20 characters length, then

11.1.1. An error message will show indicating that the provided title is too long.

11.1.2 The use case resumes at step 8.
