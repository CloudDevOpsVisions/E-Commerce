2119 - Update order status

Primary Actors

  • Vendor

Preconditions

  1. Vendor has been logged in to his account.
  2. At least one order was made to the vendor.

Basic Flow (Happy Path)

  1. Vendor navigates to list all orders page.
  2. The list all orders page is displayed.
  3. The page has a save button which is gray (unavailable).
  4. The vendor clicks on the Order status for a specific order, and changes it to completed.
  5. The save buttons becomes blue (available) indicating that there changes to be saved.
  6. The vendor clicks on the save button.
  7. The save button becomes gray again.

Postcondition

  1. The order status for the specified order is updated in the database.

Alternative Flow

None.
