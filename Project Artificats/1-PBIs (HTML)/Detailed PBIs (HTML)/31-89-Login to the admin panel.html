Title: Login to the admin panel

Tags: BWF, Private, Admin

Description:

As an admin, I want an option to login to the admin panel so that I can manage the application for different areas, also an option to remember my credentials should be there so that when I visit the admin panel next time it do not require me to re-enter credentials again for login.

Acceptance Criteria:

Action Criteria:
  • Option to remember credentials should be included.
  • Link for registration should be included.
  • Link to recover password should be included.
  • On login failure, a failure message should be shown.
  • On login success, user redirects to membership area.
