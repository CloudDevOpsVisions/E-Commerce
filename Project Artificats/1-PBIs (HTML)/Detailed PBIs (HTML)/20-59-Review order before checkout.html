Title: Review order before checkout

Tags: BWF, Public, Vistor

Description:

As a visitor, I want to review content of shopping cart along with delivery address before proceeding to checkout so that I can correct any mistake that might be there.

Acceptance Criteria:

Action Criteria:
  • Image, Title, Quantity, Price, Discount, Total amount and Gift Wrap option of added products.
  • First Name, Last Name, Email, Phone, City, State, Postal Code, Country, Address of shipping details (delivery address).
  • Option to edit shipping details and option to add/remove products from shopping cart should be included.
