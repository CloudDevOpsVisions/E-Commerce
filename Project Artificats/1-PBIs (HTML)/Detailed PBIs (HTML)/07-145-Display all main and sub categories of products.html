Title: Display all main and sub categories of products

Tags: BWF, Public, Vistor

Description:

As a visitor, I want to see all main and sub categories of products in the home page so that I can quickly select the required category.

Acceptance Criteria:

Action Criteria:
  • Display all categories in a list.
  • Display all nested category of a category in same list.
  • All displayed categories should be dynamically fetched from service provider.
  • By clicking on any category, detailed view/page of that category should be displayed.
