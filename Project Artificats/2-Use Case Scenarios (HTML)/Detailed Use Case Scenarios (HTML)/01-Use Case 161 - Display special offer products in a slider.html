161 - Display special offer products in a slider

Primary Actors

  • Visitor

Preconditions

  1. An admin have already chosen exactly three offers to display in the home page.

Basic Flow (Happy Path)

  1. Visitor navigates to the home page.
  2. A slider shows the first offer with three buttons for navigation is displayed.
  3. Visitor clicks on the second button.
  4. The second offer is displayed to the visitor.
  5. Visitor clicks on the third offer.
  6. the offer is displayed to the visitor.

Postconditions

None

Alternative Flow

None
