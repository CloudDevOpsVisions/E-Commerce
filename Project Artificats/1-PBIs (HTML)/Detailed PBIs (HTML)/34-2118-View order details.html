Title: View order details

Tags: BWF, Private, Vendor

Description:

As a vendor, I want to have a page to view a single order details so I can check any order information.

Acceptance Criteria:

Action criteria:

  • First Name, Last Name, Email, Phone, City, State, Postal Code, Country, Address of shipping details (delivery address) of the order maker should be displayed.
  • Image, Title, Quantity, Price, Discount, Total amount and Gift Wrap option of added products should be displayed.
  • Each of displayed details can be changed by clicking on it.
