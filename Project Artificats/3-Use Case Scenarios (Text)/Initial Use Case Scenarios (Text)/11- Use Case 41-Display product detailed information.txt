PBI Number	: 41

PBI Title	: Display product detailed information

Primary Actors	:
	1. Visitor

Preconditions	:
	1. A product was added by a vendor.

Basic Flow (Happy Path):
	1. Visitor navigates to the home page.
	2. Visitor clicks on a specific product.
	3. Another page for viewing the product is displayed.
	4. The page shows all information for the specified product.
	5. An add to cart option is also displayed.

	
Postconditions	:
	None

Alternative Flow:
	None
